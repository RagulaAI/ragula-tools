# langchain_tool

Python port of the langchain-tool package.

## Installation

```bash
pip install langchain_tool
```

## Usage

```python
from langchain_tool import greet_langchain_tool

# Print a greeting message
print(greet_langchain_tool())  # Output: Hello from langchain_tool!
```
